# main-site-notes

## 项目简介

`main-site-notes` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不服务于任何特定域名或单一网站，而是作为一组通用静态页面的集合，方便浏览、存储和版本管理。

这些页面可以是笔记、工具界面、信息展示页或任何其他 HTML 内容，适合那些希望以结构化方式保存和分享独立页面的场景。

## 目录说明

仓库根目录下主要包含以下内容：

- **index.html** — 仓库入口页面，提供对各个归档页面的导航或索引。
- **pages/** — 存放所有独立 HTML 页面的目录，每个页面通常以单独文件或子文件夹形式组织。
- **assets/** — 页面引用的静态资源，如 CSS 样式、JavaScript 脚本、图片等。
- **README.md** — 本文件，用于介绍仓库用途和使用方式。

> 目录结构可能根据实际归档内容进行调整，建议参考仓库内 `index.html` 或 `pages/` 下的说明。

## 页面归档说明

每个独立 HTML 页面均遵循以下原则：

- 页面之间相互独立，不依赖本仓库以外的外部资源（除非明确说明）。
- 页面内容保持简洁，不包含任何商业推广或诱导性信息。
- 归档的页面可以是完整的工具、示例、笔记或展示页，但均不针对特定网站或域名。

如需新增归档页面，请在 `pages/` 目录下创建新的 HTML 文件，并确保其内部链接和资源引用正确。

## 维护说明

本仓库由维护者不定期更新，欢迎通过提交 Issue 或 Pull Request 参与贡献。提交新页面时请遵循以下基本规范：

- 页面编码统一为 UTF-8。
- 避免在页面中包含外部跟踪脚本、广告或恶意代码。
- 尽量保持文件结构清晰，资源文件放入 `assets/` 对应子目录。
- 如有需要，可在页面内添加简短说明注释。

对于仓库中已有的页面，维护者会尽量保持其可访问性和兼容性，但不保证所有页面永久保持不变。建议使用者根据自身需求自行备份。
